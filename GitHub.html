كيفية رفع مشروعك على GitHub

اتبع هذه الخطوات البسيطة لرفع موقعك ومشاركته مع العالم

1. إنشاء حساب GitHub

إذا لم يكن لديك حساب بعد، اتبع هذه الخطوات:

  1. اذهب إلى github.com
  2. انقر على "Sign up"
  3. املأ المعلومات المطلوبة
  4. اختر الخطة المجانية (Free)
  5. قم بتأكيد بريدك الإلكتروني

2. إنشاء مستودع جديد

بعد تسجيل الدخول، قم بإنشاء مستودع جديد:

  1. انقر على زر "+" في الزاوية اليمنى العليا
  2. اختر "New repository"
  3. اختر اسمًا لمستودعك (يفضل بدون مسافات)
  4. أضف وصفًا موجزًا للمشروع
  5. اختر "Public" لجعله مرئيًا للجميع
  6. اختر "Initialize this repository with a README"
  7. انقر على "Create repository"

3. تثبيت Git على جهازك

لرفع الملفات، تحتاج إلى تثبيت Git:

  1. اذهب إلى git-scm.com
  2. انقر على "Download" لتنزيل البرنامج
  3. شغّل الملف المُنزّل واتبع خطوات التثبيت
  4. اترك جميع الإعدادات الافتراضية كما هي

4. رفع الملفات عبر Git

افتح terminal أو command prompt واتبع ما يلي:

# الانتقال إلى مجلد المشروع
cd /مسار/مجلد/المشروع

# تهيئة Git في المشروع
git init

# إضافة جميع الملفات
git add .

# حفظ التغييرات مع رسالة
git commit -m "First commit"

# إضافة رابط المستودع على GitHub
git remote add origin https://github.com/اسم المستخدم/اسم المستودع.git

# رفع الملفات إلى GitHub
git push -u origin main

5. رفع الملفات بطريقة سهلة

إذا واجهتك صعوبة مع Terminal، يمكنك رفع الملفات مباشرة:

  1. اذهب إلى المستودع الذي أنشأته على GitHub
  2. انقر على "Add file" ثم "Upload files"
  3. اسحب وأفلت مجلد مشروعك في المنطقة المخصصة
  4. أو انقر على "choose your files" لاختيار الملفات
  5. أضف رسالة توضح التغييرات (مثل: "رفع الموقع")
  6. انقر على "Commit changes"

6. نشر الموقع باستخدام GitHub Pages

لجعل موقعك متاحًا للعموم:

  1. اذهب إلى إعدادات المستودع (Settings)
  2. اختر "Pages" من القائمة الجانبية
  3. اختر "main" كفرع (Branch) ثم "/root" كمجلد
  4. انقر على "Save"
  5. انتظر بضع دقائق حتى يتم النشر
  6. سيكون موقعك متاحًا على: https://اسم المستخدم.github.io/اسم المستودع

ملاحظة مهمة: تأكد من أن الصفحة الرئيسية命名为 index.html حتى يعمل الموقع بشكل صحيح.
